# Ingress

On a high-level, Ingress is about accessing multiple web apps through a single LoadBalancer Service.

## Hands-on with Ingress

In this lab, you'll implement and learn the following steps:

1. Install the NGINX Ingress controller
2. Configure an Ingress class
3. Deploy a sample app
4. Configure an Ingress object
5. Inspect the Ingress object
6. Configure DNS name resolution
7. Test the Ingress
